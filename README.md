This is a fabric mod for minecraft 1.16.1. 
For 1.0.0 Fabric api is optional.
For 2.0.0 Fabric api and Atum is A MUST.
java21+ is required.

Generate a world with separate seeds for different dimisions.

Inspiration taken from MCSR Ranked (https://mcsrranked.com/)
